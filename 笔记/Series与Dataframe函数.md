| 函数   | Series | Dataframe | 描述                              |
| ------ | ------ | --------- | --------------------------------- |
| axes   | +      | +         | 返回行轴标签列表                  |
| dtype  | +      | +(s)      | 返回对象的数据类型                |
| empty  | +      | +         | 如果系列为空，则返回`True`        |
| ndim   | +      | +         | 返回底层数据的维数，默认定义：`1` |
| size   | +      | +         | 返回基础数据中的元素数            |
| values | +      | +         | 将系列作为`ndarray`返回           |
| head() | +      | +         | 返回前`n`行                       |
| tail() | +      | +         | 返回最后`n`行                     |
| T      | -      | +         | 转置行和列                        |
| shape  | -      | +         | 返回表示`DataFrame`的维度的元组   |
